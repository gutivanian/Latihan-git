Menggabungkan Branch

Kita telah menyelesaikan emergency fix. Selanjutnya kita dapat menggabungkan branch master dengan branch emergency-fix. Pertama-tama kita perlu pindah ke branch master dengan git checkout master. Kemudian kita dapat gabungkan branch master dengan branch emergency-fix dengan git merge emergency-fix

Merge Conflict

Ketika di branch master dan branch emergency-fix memiliki file yang sama dan memiliki isi yang berbeda, maka akan muncul merge conflict saat dilakukan penggabungan. Untuk menyelesaikan masalah ini kita bisa memilih salah satu dari kedua versi file yang ama terebut. Untuk memilih buka code editor dan piih salah satu. Setelah selesai dipilih, dapat dilakukan staging dan commit ke branch master.
